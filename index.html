Software

This project has two main software components:

  • Arduino Code: Controls sensors, LEDs, and button input.

Example Arduino logic:

void loop() {
    int tempValue = analogRead(tempPin);  // Temperature sensor value
    int heatValue = analogRead(heatPin);  // Heat sensor value

    if (tempValue > tempThreshold) {
        digitalWrite(ledGreen, HIGH);
        digitalWrite(ledRed, LOW);
    } else if (tempValue < tempThreshold) {
        digitalWrite(ledGreen, LOW);
        digitalWrite(ledRed, HIGH);
    }

    if (heatValue > heatThreshold) {
        digitalWrite(ledGreen, HIGH);
        digitalWrite(ledRed, LOW);
    } else if (heatValue < heatThreshold) {
        digitalWrite(ledGreen, LOW);
        digitalWrite(ledRed, HIGH);
    }

    if (tempValue > tempThreshold && heatValue > heatThreshold) {
        digitalWrite(ledGreen, HIGH);
        digitalWrite(ledRed, LOW);
    } else if (tempValue < tempThreshold && heatValue < heatThreshold) {
        digitalWrite(ledGreen, LOW);
        digitalWrite(ledRed, HIGH);
    }
}
